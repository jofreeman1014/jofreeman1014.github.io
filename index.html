World Weather Trends

It is well known that certain weather patterns across the globe are related to the latitude where the measurement was taken. This project focused on observing different weather and climate patterns across the globe, and seeing which weather parameters are affected by latitude, and how. The data was acquired using OpeanWeatherMap's API feature to pull live data from weather stations around the globe. The data is then plotted against latitude to seek out which parameters are affected by latitude.

Tools and Techniques:
  • Python (Jupyter Notebook, Matplotlib, Pandas)
  • OpenWeatherMap API
  • Hosted through Github Pages

Belly-Button Biodiversity

Using data from the Rodunn Lab's "Belly Button Biodiversity" dataset, I built a single-page application dashboard to explore the data, listing subject information alongside bacteria colony information. The dashboard contains 3 visualizations: Top 10 samples present in sample set as percentages, a color-coded bubble chart that displays each sample's bacterial operational taxonomic unit (OTU) plotted against the sample values, and a gauge to display each subject's bellybutton washing frequency (wfreq). This dashboard is fully interactive, so users can see each of the charts update automatically when selecitng a new subject from the dropdown on the page.

Tools and Techniques:
  • HTML & CSS
  • Javascript
  • Python (Flask)
  • Hosted through Heroku

Office Mart Retail

This Tableau visualization is a single-page dashboard that displays Office Mart retail data across Europe. This viz is interactive, and updates when users select a location on the provided map. It then refreshes the other two plots automatically. This dashboard includes a Sales/Profit bubble chart and a Sales/Target area chart divided into Furniture, Office Supplies, and Technlolgy sales. There is also a year slider to parse data for specific years, or for all years within the dataset.
